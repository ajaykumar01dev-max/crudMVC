# 🗂️ CRUD MVC Application

A simple **CRUD (Create, Read, Update, Delete)** application built using the **MVC (Model-View-Controller)** architecture.

---

## 🚀 Features

- ➕ Create new records
- 📖 Read/View data
- ✏️ Update existing records
- ❌ Delete records
- 🧱 Structured using MVC pattern

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js / Express *(or your backend)*
- **Database:** MongoDB / MySQL *(update as per your project)*
